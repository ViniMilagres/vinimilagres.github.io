# Como aplicar a atualização do site

## Resumo

Este pacote contém os arquivos atualizados do site profissional:

- index.html
- curriculo.html
- historia.html
- certificados.html
- assets/css/style.css
- assets/certificados/README.md

## Passo a passo recomendado com GitHub Desktop

1. Abra o GitHub Desktop.
2. Clique em File > Clone repository.
3. Escolha o repositório: ViniMilagres/vinimilagres.github.io.
4. Clone para uma pasta local.
5. No GitHub Desktop, clique no seletor de branch, onde aparece "Current branch".
6. Clique em "New branch".
7. Nome sugerido: atualizacao-site-curriculo-certificados.
8. Confirme que a branch foi criada a partir da main.
9. Extraia este pacote ZIP dentro da pasta do repositório clonado.
10. Permita substituir os arquivos existentes quando o Windows perguntar.
11. Volte ao GitHub Desktop.
12. Confira os arquivos alterados na aba Changes.
13. Escreva a mensagem de commit:
    Atualiza site profissional com currículo, história e certificados
14. Clique em Commit to atualizacao-site-curriculo-certificados.
15. Clique em Publish branch.
16. Depois clique em Create Pull Request.
17. Revise no GitHub.
18. Se estiver tudo certo, faça Merge para a main.

## Importante sobre certificados

A página certificados.html já está preparada para apontar para a pasta:

assets/certificados/

Por segurança, publique apenas versões dos certificados sem CPF, RG, endereço completo, QR Code sensível ou dados pessoais desnecessários.

Os nomes sugeridos para os arquivos estão no README dentro da pasta assets/certificados/.
