# Certificados

Coloque nesta pasta os arquivos de certificados em PDF ou imagem.

Recomendação de segurança:
- Não publicar CPF, RG, endereço completo ou QR Code sensível.
- Usar nomes simples, sem acentos e sem espaços.
- Exemplos:
  - mba-inteligencia-artificial.pdf
  - banco-de-dados-sql-modelagem.pdf
  - python-introducao.pdf
  - delphi-xe8.pdf
  - tic-senai.pdf
